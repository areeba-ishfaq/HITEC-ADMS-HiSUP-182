# HITEC-ADMS-HiSUP-182

HITEC Smart University Portal - ADBMS Project



\## Database Tables



| Table | Description |

|-------|-------------|

| Departments | University departments (CS, Business, Engineering) |

| Programs | Degree programs (BSCS, BBA, BSEE) |

| Students | Student personal and academic information |

| Faculty | Teacher/Professor employment details |

| Courses | Course catalog with codes and credit hours |

| Sections | Course sections for specific semesters |

| Enrollments | Student course registration records |

| FeeStructure | Fee amounts for each program per semester |

| FeePayments | Student fee payment transactions |
